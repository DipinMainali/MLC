# Prajesh Web

Next.js 15 portfolio site with a custom PostgreSQL backend, Auth.js JWT sessions, and modular feature-based services.

## Start here

- `docs/SETUP.md` — install and run locally
- `docs/API.md` — API reference and response format
- `docs/DATABASE.md` — schema and relationships overview
- `docs/ADMIN.md` — non-technical content editing guide
- `docs/CODEBASE_MEMORY.md` — living architecture notes for ongoing work

## Core stack

- Next.js App Router
- PostgreSQL on Aiven
- Prisma ORM
- Auth.js / NextAuth JWT sessions
- Google and Facebook OAuth
- Zod validation and consistent JSON API responses
